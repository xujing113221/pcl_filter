Jabref Settings
===============


Jabref-Version
--------------

* Version >= 2.10


Saving settings
---------------

* Open JabRef.
* Switch to Option->Preferences.
* Open tab File:
    - Under Sort order choose: Save ordered by author/editor/year
    - Under Field saving options: deselect all the three bullets.


BibTeX settings
---------------

* Open JabRef.
* Switch to Options->Preferences.
* Open tab BibTeX key generator:
    - Choose as Default pattern: [auth][auth3_2][auth3_3][auth3_4][year]
    - At the bottom select: Generate keys before saving


Char encoding of database
-------------

* Jabref menu File->Database Properties
* Encoding: UTF8
